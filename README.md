# NourAI : your recovery companion 
A multi-agent AI companion for eating disorder recovery. 
Built with Qwen, FLUX, AMD GPU, and Gradio. 

## Setup
1. Clone the repo
2. Copy `.env.example` to `.env` and fill in your keys
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python ui/gradio_app.py`

## Disclaimer
Nourish is a creative companion, not medical advice.
Always work with your care team.
